# TextSpeedReader

TextSpeedReader 是一款基於 .NET Windows Forms 開發的多功能文字閱讀與編輯器，旨在提供流暢的檔案瀏覽與多樣化的文字處理工具。

## 專案用途

本工具主要用於快速開啟、閱讀並編輯各類文字檔案。除了基本的編輯功能外，它還整合了檔案管理、編碼修正以及繁簡轉換等進階功能，特別適合開發者、文字工作者或需要處理不同編碼檔案的使用者。

## 主要功能

### 檔案瀏覽與管理
- **樹狀目錄 + 檔案清單**：TreeView 顯示目錄結構（惰性載入），ListView 顯示當前目錄內的檔案。
- **多格式支援**：可在編輯器中開啟 `.txt`, `.cs`, `.yaml`, `.js`, `.py`, `.md`, `.css`, `.json`；以 WebBrowser 預覽 `.html`, `.htm`。
- **拖曳開啟**：支援從 Windows 檔案總管拖曳檔案或資料夾至程式視窗。
- **資料夾操作**：新增、刪除（移至資源回收筒）、重新命名，亂碼修正，繁簡轉換。
- **檔案操作**：支援檔案拖曳搬移/複製至上方目錄區，以及新增、刪除（移至資源回收筒）、重新命名，單檔與多檔批次更名皆支援，繁簡轉換。
- **檔名搜尋**：在 ListView 中以關鍵字搜尋並選取符合的檔案。

### 文字編輯
- **RichTextBox 編輯器**：支援字型選擇、字體大小調整（預設尺寸清單）、自動換行切換。
- **尋找 / 取代**：支援「區分大小寫」與「全字匹配」，可循環查找。
- **拖曳選取文字**：可拖曳外部文字直接插入。

### 文字處理工具
- **移除多餘斷行**：合併段落內的單行換行，保留雙換行（段落分隔）。
  - 進階版：可保留以句號、問號、驚嘆號結尾的行換行。
- **確保段落空白行**：讓每對非空行之間恰好有一個空白行。
- **移除空白行**：移除所有空白行，使各行緊密排列。
- **移除行首/尾空白**：清除半形空格、TAB、全形空格。
- **行首加縮排**：為每個非空行加上指定數量的半形空格（數量可在設定中調整）。
- **行尾補句號**：對未以標點結尾的行自動補上全形句號「。」。
- **行排序**：正向 / 反向排序，支援保留空白行的分段排序模式。
- **依判定字串拆行 / 合行**：可在設定中指定「新行開頭判定字串」與「新行結尾判定字串」，對文字進行段落切分或合併。
- **插入行首/尾標記**：為每個非空行加上設定中指定的開頭與結尾字串。
- **行號標記插入**：在選取範圍的每一行行首插入「AnnotationBegin + 絕對行號 + AnnotationEnd」，已有標記時自動更新行號。

### 編碼修正
- **智慧編碼修正（ReCodeText）**：將目前文字以「錯誤編碼 → 正確編碼」重新解碼，預覽多種組合後選取。支援可選同步執行簡→繁轉換。
- **檔名編碼修正（ReCodeFileName）**：修正 ListView 中選取的檔案或目錄名稱的編碼錯誤。
- **目錄名稱編碼修正（ReCodeFolderName）**：修正 TreeView 中選取目錄的名稱編碼。
- **批量編碼修正（ReCodeFullFoldersFilesName）**：遞迴修正指定目錄下所有檔案與子目錄的名稱編碼。
- 支援的編碼組合：UTF-8、UTF-8-BOM、GBK（936）、Big5（950）、GB18030（54936）、Unicode（UTF-16 LE）。

### 繁簡轉換
- **文字內容轉換**：將目前開啟的 `.txt` / `.json` 轉換為簡體並另存（檔名加 `_簡體1`）。
- **批次轉繁/轉簡**：對 ListView 中選取的多個 `.txt` / `.json` 批次轉換並另存。
- **檔名繁簡轉換**：將 ListView 中選取的檔案或目錄名稱以 OpenCCNET 轉為繁體。
- **目錄名稱繁簡轉換**：將 TreeView 中選取目錄的名稱轉為繁體。
- 使用 OpenCCNET 函式庫（`HantToHans` / `HansToTW`）。

### 儲存功能
- **覆蓋儲存**：以原始編碼覆蓋目前檔案。
- **另存新檔**：以 UTF-8-BOM 儲存，支援多種副檔名。
- **選取內容另存**：將目前選取的文字另存為新檔，自動產生不重複的三位數序號檔名。
- **整篇分割儲存**：將整篇文字以每份 3,000 字切割，批次存為多個序號檔案。

### HTML 瀏覽（WebBrowser 模式）
- 自動偵測 HTML 檔案編碼（GB2312/GBK → UTF-8），必要時產生臨時檔案確保正確顯示。
- 可調整頁面縮放比例（預設縮放清單）。
- 複製 WebBrowser 選取文字並轉換為繁體/簡體後儲存為 `.txt`。

### 閱讀紀錄
- 自動儲存最近閱讀的檔案路徑與閱讀字元位置，方便續讀。
- 支援「自動開啟上次目錄」。
- 最近目錄與最近檔案各保留最多 10 筆（MRU 順序）。

## 技術架構

| 項目 | 內容 |
|------|------|
| 框架 | .NET Windows Forms |
| 語言 | C# |
| 主要表單 | `FormTextSpeedReader`（分拆為多個 partial class 檔案） |
| 設定儲存 | INI 格式，`.\TextSpeedReader_Settings.ini` |
| 編碼偵測 | UDE（Universal Charset Detector）+ BOM 判斷 |
| 繁簡轉換 | OpenCCNET |
| 檔案刪除 | `Microsoft.VisualBasic.FileIO`（使用資源回收筒） |
