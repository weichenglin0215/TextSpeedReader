# TextSpeedReader

TextSpeedReader 是一款基於 .NET Windows Forms 開發的多功能文字閱讀與編輯器，旨在提供流暢的檔案瀏覽與多樣化的文字處理工具。

## 專案用途

本工具主要用於快速開啟、閱讀並編輯各類文字檔案。除了基本的編輯功能外，它還整合了檔案管理、編碼修正以及繁簡轉換等進階功能，特別適合開發者、文字工作者或需要處理不同編碼檔案的使用者。

### 主要功能
- **檔案瀏覽系統**：透過樹狀目錄 (TreeView) 與檔案清單 (ListView) 快速定位檔案。
- **多格式支援**：支援 `.txt`, `.cs`, `.yaml`, `.js`, `.py`, `.md`, `.css` 等文字格式，以及 `.html`, `.htm` 網頁預覽。
- **智慧編碼修正**：提供專用的編碼修正視窗，協助使用者修復因編碼錯誤導致的亂碼，支援 UTF-8 (含 BOM), GBK, Big5, Unicode 等常見編碼。
- **繁簡轉換**：支援文件內容或檔案名稱的繁簡中文轉換。
- **批次處理**：可批次修正目錄下所有檔案的名稱編碼。
- **AI 輔助支援**：整合 AI 生圖提示詞編輯與優化功能（自動換行、逐行排序等）。
- **閱讀紀錄**：自動儲存最近閱讀的檔案路徑與閱讀位置，方便續讀。

## 更新紀錄

### V2.7.0.0 -2026-05-09
- **強化編碼修正功能 (ReCodeText)**：
    - 實作了針對目前編輯中文字的編碼修正功能。
    - 支援選取部分文字作為預覽樣本，以便精確判斷原始編碼。
    - 加入「自選編碼重新命名」視窗，提供直觀的編碼轉換預覽。
- **擴展 UTF-8-BOM 支援**：
    - 在所有編碼轉換與預覽邏輯中加入 `UTF-8-BOM` 格式。
    - 確保檔案儲存時能正確寫入 UTF-8 識別標籤。
- **支援更多程式碼與文件格式**：
    - 將 `.js`, `.py`, `.md`, `.css` 加入支援列表。
    - 更新檔案瀏覽器邏輯，使上述格式能正確開啟於編輯器中而非僅顯示於清單。
    - 更新儲存對話框的過濾器 (Filter)，讓使用者能以正確的副檔名儲存檔案。
- **優化檔案類型管理**：
    - 重構 `FileSystemManager` 與 `FormTextSpeedReader` 之間的連動，使新增支援格式更加容易且統一。
